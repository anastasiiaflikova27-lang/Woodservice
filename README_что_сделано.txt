Вуд Сервис — финальная версия сайта с правками от 05.06

Что изменено:
- название везде заменено на «Вуд Сервис» без «РФ»;
- в шапку вставлен новый логотип в стиле выбранного варианта: знак из ламелей + название + подпись «производство изделий из дерева · СПБ»;
- логотип сделан вектором в HTML/CSS, поэтому он не мылится на MacBook/Retina и хорошо масштабируется;
- максимум два шрифта на сайте: Prata для крупных заголовков и Golos Text для текста, меню, кнопок и интерфейса;
- в контактах исправлено «Написать Максу»: теперь есть «Написать в приложение MAX», «Написать в Telegram» и «Позвонить»;
- добавлены ссылки MAX / Telegram / телефон в футерах и в нижней фиксированной панели;
- убраны все упоминания «Вуд Сервис РФ» из текстов, title, description и JSON-LD;
- оставлена отдельная страница портфолио: portfolio.html;
- оставлена отдельная страница под Яндекс-рекламу: zhalyuziynye-fasady.html;
- сохранены карточки работ, фильтры портфолио, модальные окна с фотографиями;
- добавлено мобильное меню, чтобы навигация не пропадала на телефоне;
- поправлены адаптивные стили для телефона, планшета и MacBook: шапка, логотип, hero, сетки фото, портфолио и нижние кнопки не должны ломаться.
